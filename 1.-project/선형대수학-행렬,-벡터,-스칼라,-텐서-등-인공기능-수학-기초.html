벡터와 행렬의 의미

    • 02:26: 벡터는 1줄로 나열된 숫자들의 묶음이며,

데이터의 (차원의) 종류

  • 03:58
    • Point: 점. [0, 1]. 있거나 없거나.
    • Scalar: 1차원. 수직선 하나. 숫자 자체. 1, 2, 3, 등.
    • Vector: 2차원. 수직선 여러 개. 수직선이 3개라 입체적으로 보이더라도, 해당 데이터는 (1, 0, -2)와 같이 일직선으로 표현될 수 있으므로 벡터인 것. 숫자들의 묶음. [1, 2, 3] 이런 식.
    • Matrix: 2차원 벡터. 그러니까, 벡터가 여러 개 나열되어 있는 것.
    • Tensor: Matrix의 2차원 버전?, 2차원 매트릭스. 혹은 그 이상.
  • 13:07: 표현들
    • 16:13: 차원에 대해 직관적으로 이해 가능한 표현

  2. 벡터의 Linear Independent(선형 독립)
    • 18:26선형 독립이란 각각의 벡터가 얼만큼 개성이 있느냐에 대한 것.
    • 19:06특정 벡터는 고유의 스칼라 공간(수직선, )을 가지고 있다고 할 수 있는데, 두 개의 벡터가 가지는 두 개의 수직선만을 이용하여 전체 평면상의 영역을 표현할 수 있는가가 선형 독립과 관련된다.20:43
      • 24:28여기서 알게되는 흥미로운 사실은, n차원 공간을 정의()하기 위해서 벡터 n개가 필요하다는 것이다. 이때 각각의 벡터를 라고 하며, 'n개의 벡터, 즉 basis들이 n차원 공간을 span할 수 있다'고 부른다.
    • 21:55선형 독립의 수학적 표현
      • 22:50여기서 벡터의 덧셈이 정의가 됨

2. 행렬

  1. Matrix as Vector Space, Rank
    • 25:31: 행렬은 n개 벡터()들의 묶음이며, n개의 벡터들은 서로 선형 독립일 경우 Basis로서 n차원 공간을 span할 수 있으며, 이때 이 행렬의 Rank를 n이라고 한다. 즉, Rank는 Matrix를 구성하는 Vector들이 Span할 수 있는 최대 n차원 공간을 의미한다.
  2. Matrix as Linear Operator
    • 41:43벡터에 행렬을 곱하여 벡터가 나온다는, 36:10의 내용의 결론.
    • 즉, 행렬을 곱한다는 것은 어떤 벡터를 다른 벡터로 이동시킨다는 의미.
    • 44:00왜냐하면면, 기준축(축, 축)을 특정 행렬을 이루는 열벡터로 바꿔 생각하게 되기 때문에. 즉, 좌표공간의 변형47:46, 따라서 원래의 벡터가 이동하는 것처럼 보임.
    • 49:01예 - Rotation Matrix

벡터와 행렬, 즉 선형대수학에서의 덧셈(뺄셈)과 스칼라곱

  • 36:10어떤 벡터()와 행렬(⨉)을 곱하면 벡터()가 나온다?
    • 36:24행렬의 곱셈에서 차원에 대해; 이것이 이유.
    • 37:05그런데 이러한 것이 어떤 의미를 가지는가?
      • 어떤 행렬 가 있다고 하자. 이때, 행렬 에 곱해져서 0이 되는 또 다른 벡터 가 있다고 하자. 즉 일 때, 가 만드는 수직선상의 공간을 Range Space라고 하고, 를 Null Space라고 한다.
      • 이때 Null Space가 존재한다는 것은 원래 행렬 에 선형 독립이지 않은, 즉 서로 종속인 열벡터가 있었다는 것이고, 그래서 Rank가 원래 응당 가져야할 값보다 떨어졌었구나 판단할 수 있음.
